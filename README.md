# 哈工大研究生 LaTeX 实验报告模板

这是一个可复用的中文实验报告 LaTeX 模板，根据哈工大研究生院提供的 Word 模版制作而来。

模板已包含一份完整的示例报告结构，覆盖封面、目录、标题层级、公式、算法伪代码、表格、图片、子图、交叉引用、代码片段和参考文献等常用报告元素。使用者可以直接在 `report.tex` 中替换课程信息、正文内容、图片和参考文献条目。

## 文件说明

- `report.tex`：实验报告主文件，包含各类常用排版对象示例。
- `hitrep.cls`：报告模板类文件，定义封面、页眉页脚、标题、图表、算法和参考文献等格式。
- `citations.bib`：参考文献示例文件。
- `hitrep-gbt7714-numerical.bst`：参考文献样式文件。
- `Fig/image.png`：示例图片，可替换为自己的图片。
- `report.pdf`：模板编译后的 PDF 示例。

## 使用方式

1. 修改 `report.tex` 开头的课程名称、院系、专业、姓名、学号和学生类别。
2. 按需替换正文中的示例标题、公式、表格、图片、算法、代码和参考文献。
3. 将自己的图片放入 `Fig` 文件夹，并在 `report.tex` 中修改图片路径。
4. 在 `citations.bib` 中维护参考文献条目，并在正文中使用引用键。

模板默认会根据编译日期自动生成封面的学年和学期；如需手动指定，可在 `report.tex` 开头取消对应注释并填写。
